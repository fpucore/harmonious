CMDC (Centrally Managed Digital Currency)


CMDC is an acronym for Centrally Managed Digital Currency and can be stylized as $CMDC.

It is a prototype digital currency (or asset) with a specification which pegs it to a defined existing currency to determine its value. Due to the way it has been developed, it is inherently flawed. The original value rate was proposed to be 1:1.25. For example, 1 US dollar would equate to 1.25 CMDC. However, big development changes to the mining process have now altered the original rate. The value rate is now 1:2500.

Bitcoin is infamous for being energy and resource intensive during the mining process. Some analysts believe that its high rate of energy use could see its eventual demise. CMDC has been designed specifically to be light on system resources for both mining and storage. This helps increase energy efficiency and ease the storage burden while still being able to mine large amounts. During the mining process it may push all cores of the central processing unit to maximum but memory usage remains moderate. While there is no technical limitation to the total number of CMDC blocks that can be put into circulation, there are still restrictions of the amount of available blocks in holding in the social distribution vault.

CMDC is mined using a parallel processing and file duplication combination. Mined blocks are created from a raw block, called "block0". To prevent fraudulent duplication of blocks, each individual block is dated and timestamped which cannot be altered through traditional file duplication methods of the operating system. Once the mining process is complete, mined blocks are then refined. Without being refined, the blocks are worthless. During the refining process the blocks are compressed, encrypted and digitally signed with a unique and secret key which is not made public. It is this secret key which is used to identify authenticity of CMDC block sets.

Once blocks have been mined, refined, encrypted, digitally signed and verified, they are moved into a social distribution vault for circulation. We are currently testing software which enables buying of CMDC and verification of transactions. We are still developing software which enables selling and trading. CMDC operates on our private blockchain, called Gobble Chain.

One area of development focus is on integrating support for non-fungible tokens[?], or NFTs. The specification for NFT integration with our blockchain is yet to be finalized and more information will be provided when available. We aim to offer NFT integration which will be applicable to original creations in the forms of digital images, photos, audio, video and other digital artwork creations, which can be sold as NFT by rights holders and bought using CMDC. The proof of rights and the transaction will be integrated into Gobble Chain.
