Components


Mainframe
The critical development system which runs core software which enables Harmonious Platform to function.
LinkNet
The core publishing component providing network interoperability.
Soul
The primary component which maintains system uptime of the network.
KEEP UP!
The secondary component which provides support to Soul through Master/Slave redundancy to maintain system uptime of the network.
Heartbeat
The component which maintains version control of all critical components.
LinkNet Status Monitor Tool
The component which maintains Online/Offline status of all critical components.
Pouch
A temporary framework to accomodate the viADDR database and Pitrix.
1V
A file-based operating sub-system currently under heavy development. Once implemented, components currently integrated into the Pouch framework will be migrated into 1V.
viADDR database
Comprised of two components - the main database module and a proxy module, and written in its own programming language - viADDR is a framework used for distribution of the Pitrix viADDR Global Allocation table.
Pitrix
The component which provides global virtual office functions and session authentication.
