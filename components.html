Components


Harmonious Platform Systems is completely transparent about our operations and components. We deploy a large amount of free and open-source software (FOSS) which is used in combination with proprietary components we have developed to fill the gaps, unite systems and to guarantee interoperability. Although we do not share the source code for these in-house developed components, we are completely open and transparent about how all components function and what purpose they serve to Harmonious Platform.
We are dedicated supporters of FOSS, Linux and BSD.


Inter-Com Net
Primary component which provides unified network functionality to all other system components.
Mainframe
Core component which enables Harmonious Platform to function.
LinkNet
Core publishing component.
LinkNet (front-end)
Front-end for the core publishing component.
LinkNet (back-end)
Back-end for the core publishing component.
Watchtower
Low-level security gateway for LinkNet Front-end.
Soul
Primary component which maintains system uptime.
KEEP UP!
Secondary component which provides support to Soul, by offering Master/Slave redundancy to maintain system uptime.
Heartbeat
Component which maintains version control of critical components.
LinkNet Status Monitor Tool
Component which monitors Online/Offline status of critical components.
M2M web syntax
Proprietary internal web syntax compiler and interpreter used for web publishing.
Wingspan
Verified search and publishing platform that runs parallel to LinkNet.
Churn Resolver
An alternative, proprietary domain name allocation system.
Pitrix
Component which provides global virtual office functions and session authentication.
Harmonious Access Browser
Web browser configured specifically to function with Harmonious Platform.
Project Management/Redmine
Component which provides complete project management function.
Git/Issue Tracker/Bug Tracker/Q & A
Main suite of components which provide Git source code management, issue and support tracking, bug tracking and Question & Answer platform.
Email System
Internal mail system configured with local email server and client.
Dark Unified Messaging Portal (DUMP)
Unified internal message board platform.
HRM/CRM
Configuration of two components which provide human resource management and customer relationship management.
Centrally Managed Digital Currency (CMDC)
Proprietary digital currency with full technical compliance with Harmonious Platform.
